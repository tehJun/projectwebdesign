How to Read Tabs

What are Tabs?

Tabs are a form of musical notation associated with stringed instruments. When learning guitar it is important that you are familiar and comfortable with reading tabs. Don't worry though, learning how to read tabs is an easy endeavour, but it is important to get right. Tabs have one horizontal line for each string of the instrument. There are numbers on this string that represent what fret should be pressed in when playing. See below for an example.

Tab

In this example we can see numbers of each line. These numbers represent what fret of the guitar we will be playing. The number 0 indicates that we will be playing the open string, so we will be playing that string without pressing in on any frets. In other examples we can see different symbols being used in the tabs, each representing a different technique to incorporate into your playing.
